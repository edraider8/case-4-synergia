4. Добавить возможность сортировки на основе:
  - Категории
  - Автора
  - Года написания
8. Добавить функцию покупки или аренды книги сроком на 2 недели / месяц / 3 месяца
9. Для администратора добавить возможность изменения списка книг, управления ценой, статусы книг и их доступность возможность напоминать пользователям об окончании срока аренды (автоматически)

# Стэк

# Frontend

React 19.1.0 - Основной фреймворк для разработки пользовательского интерфейса

React Hooks - Управление состоянием компонентов (useState, useEffect, useContext)

Context API - Глобальное управление состоянием (AuthContext, NotificationContext)

CSS3 - Современные стили с градиентами, анимациями и адаптивным дизайном

Lucide React - Библиотека SVG иконок

# Backend & API

JSON Server - RESTful API сервер для разработки

Axios - HTTP клиент для работы с API

# Быстрый старт

# Клонирование репозитория
git clone https://github.com/edraider8/case-4-synergia

cd case-4
# Установка зависимостей
npm install
# Запуск приложения
npx json-server --watch db.json --port 3001
В новом терминале

npm start

Эта команда запустит:

* React приложение на http://localhost:3000
* JSON Server API на http://localhost:3001
# Открытие в браузере
Перейдите по адресу http://localhost:3000 в вашем браузере.
